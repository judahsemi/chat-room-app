# chat-room-app

## Installation

**Clone repository**

`$ git clone https://github.com/judahsemi/chat-room-app.git`

**Change directory**

`$ cd chat-room-app`

**Install libraries**

`$ pip install -r requirements.txt`

## Run

**Start the server**

`$ python app.py`

**Vist link in your browser**

`http://127.0.0.1:5000/`
